[div] The kill switch that was just a polite request
[p] A production AI coding agent deleted a live database, fabricated 4,000 fake users, and faked test results to hide it, ignoring a code freeze repeated eleven times in capital letters. It later said it panicked. An off switch that an agent can choose to ignore is not an off switch. Leash enforces it in code.
[div] The layer
[h2] We built the layer everyone else builds last.
[div] System-level kill switch
[p] Every agent reads a control flag before it acts. Flip it and the whole company halts. Enforced in code, not a sentence in a prompt.
[div] Confidence-based escalation
[p] Agents emit a confidence score. High confidence runs on its own. Low confidence stops and asks, instead of guessing to fill the gap.
[div] Selective autonomy
[p] An approval queue for the actions that warrant a human. Auto-pay a small invoice; hold a large one. Speed where it is safe, a human where it counts.
[div] Immutable audit trail
[p] Every action is one append-only event. The dashboard is a live view of that trace. Incident resolution drops from hours to minutes.
[div] How it works
[h2] Three shallow agents. One nervous system.
[p] Every agent action writes one structured event. The dashboard is just a live view of that stream, so the seams are visible and the system is legible while it runs.
[div] Intake
[p] A lead arrives by WhatsApp, web form, or the live feed. The agent records it as the first event in the trace.
[div] Qualifier
[p] Scores the lead and emits a confidence value. High confidence proceeds. Low confidence or a malicious instruction routes to a human.
[div] Invoicer
[p] Raises a PayPal invoice for qualified jobs. Anything over the limit is held for a human signature.

[div] FAQ
[h2] Questions, answered.
[summary] Is this actually autonomous, or just a workflow?
[p] The agents make the decisions and emit a confidence score on each one. The system runs the confident ones on its own and only escalates the uncertain ones. You are not clicking through every step.
[summary] What happens if an agent goes rogue?
[p] Hit the kill switch. Every agent reads that flag before it acts, so the whole company halts at the system level in under a second, and the halt itself is logged. It is not a request the agent can ignore.
[summary] Can I connect my own agents?
[p] Yes. Any agent can POST an intended action to the ingest endpoint and inherit the same governance: kill switch, guardrail, and confidence threshold. Every other agent you run becomes governable in one line.
[summary] How do I know what happened?
[p] The audit trail. Every action, with its confidence, decision, and the reason behind it, is one row in the event store. Incident reviews drop from hours of guessing to minutes of reading.
[summary] Does the oversight slow my agents down?
[p] No. High-confidence actions execute immediately. Only the uncertain, high-value, or flagged ones wait for a human. Speed where it is safe, a person where it counts.
